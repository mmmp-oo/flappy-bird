# Flappy Bird - Python Edition 🐤

A simple clone of the classic Flappy Bird game developed using Python and Tkinter.

## 📷 Screenshots
(https://github.com/mmmp-oo/flappy-bird/blob/main/image.png)
## 🎮 Gameplay
- Use **Spacebar** or **Left Click** to make the bird jump.
- Avoid hitting the top and bottom pipes.
- Score increases every time you pass a set of pipes.
- Game Over appears if the bird hits an obstacle or falls out of bounds.

## 🔧 Features
- Gravity and jump physics.
- Score and high score system (saved in `record.txt`).
- Simple graphics using `PIL` and `Tkinter`.
- Works on Windows with Python 3.x installed.

## 📁 How to Run

1. Install Python 3 if you haven't already.
2. Install the Pillow library:
   ```
   pip install pillow
   ```
3. Download all the files (`main.py`, `bg.png`, `bird.png`, `pipe.png`, `pipeup.png`)
4. Run the game:
   ```
   python main.py
   ```

## 📄 Files
- `main.py` - main game script.
- `bg.png` - background image.
- `bird.png` - player character.
- `pipe.png` - bottom pipe.
- `pipeup.png` - top pipe.
- `record.txt` - stores the best score.
